# Splunk-SIEM-Lab
I built a Splunk SIEM home lab to gain hands-on blue team experience. I hosted Splunk in a Linux KVM, forwarded pfSense firewall logs via syslog-ng, and deployed Universal Forwarders on an Active Directory Domain Controller and Windows client. Using SPL, I built alerts to detect Nmap scans, brute-force attacks, and rogue admin creations.
